# 手順
* git clone git@github.com:masarufuruya/middleman_template.git
* bundle install --path vendor/bundle
* bundle exec middleman server
* http://localhost:4567/ にアクセスしてサイトが立ち上がればOK

# 用意されている機能
* livereload
* sassコンパイル
* slimテンプレート
